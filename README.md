# Тестовое задание на должность "Девопс разработчик"

## Usage

Перейдите на [опубликованную страницу][github-pages], чтобы потестить
приложение.

Но перед этим нужно будет создать инстанс в
[консоли GREEN-API](https://console.green-api.com) по
[официальной инструкции](https://green-api.com/docs/before-start).

Далее заполните поля
[значениями](https://green-api.com/en/docs/cabinet/settings), которые вы
получили в [консоли GREEN-API](https://console.green-api.com) после создания
инстанса и с помощью кнопок на странице начните отправлять
**сообщения**/**файлы**.

[github-pages]: https://arut-plus.github.io/green-api-devops-test-assignment
